# تعلم تطوير الواجهات الأمامية (Front-End)

موقع تعليمي متكامل لتعلّم **HTML و CSS و JavaScript و APIs** بتصميم **Material Design 3** وواجهة عربية (RTL).

## 📚 المنهج الدراسي

### HTML — هيكلة الموقع (6 دروس)
- أساسيات الويب وكيف يعمل المتصفح
- العناصر والنصوص (العناوين، الفقرات، التنسيق)
- الروابط والوسائط (الصور، الفيديو، الصوت)
- القوائم والجداول
- النماذج والمدخلات
- HTML5 الدلالي (Semantic HTML)

### CSS — التصميم والتنسيق (8 دروس)
- المحددات الأساسية وطرق الربط
- نموذج الصندوق (Box Model)
- الألوان والخلفيات والتدرجات
- تنسيق النصوص والخطوط
- Flexbox
- CSS Grid
- التصميم المتجاوب (Responsive Design)
- الحركات والتأثيرات (Animations & Transitions)

### JavaScript — التفاعل والمنطق (7 دروس)
- المتغيرات وأنواع البيانات
- التحكم في التدفق (if/else, loops)
- الدوال (Functions)
- المصفوفات والكائنات (map, filter, reduce)
- التعامل مع DOM
- الأحداث (Events)
- البرمجة غير المتزامنة (Promises, async/await)

### APIs — جلب البيانات (6 دروس + مثال تطبيقي)
- ما هو API
- REST و HTTP Methods
- JSON
- Fetch API (GET/POST)
- Async/Await و أفضل الممارسات
- مثال تطبيقي: جلب بيانات مستخدم GitHub

## 🎨 التقنيات المستخدمة

- **Material Design 3** — ألوان داكنة (Dark Theme)، ظلال، زوايا دائرية
- **Roboto / Roboto Mono** — خطوط حديثة
- **RTL** — واجهة كاملة بالعربية
- **Keyboard Navigation** — تنقل بالأسهم (← →)
- **CSS Grid & Flexbox** — تخطيط متجاوب
